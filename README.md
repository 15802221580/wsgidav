# ![logo](logo.png?raw=true) WsgiDAV

WsgiDAV is a generic [WebDAV](http://www.ietf.org/rfc/rfc4918.txt) server 
written in Python and based on [http://www.python.org/dev/peps/pep-0333/ WSGI].
